## 优化原则

1. **稳定性优先** — 每次优化后必须通过 `python -m pytest tests/ -v`
2. **渐进式** — 一次只优化一张或一组牌，避免大规模改动
3. **向后兼容** — 不修改渲染逻辑（`renderer.py`），只修改牌面数据文件
4. **可回滚** — 每完成一个阶段，提交一次 git commit

### 阶段 1：愚者 (The Fool) — 门面优化

**优先级：** ⭐⭐⭐ 最高
**原因：** 启动测试默认展示愚者，是用户第一印象
**文件：** `tarot/cards/fool.py`

**优化方向：**
1. 增加颜色层次（当前10色 → 16色）
2. 丰富细节（眼睛、表情、服饰纹理）
3. 改善背景（悬崖、天空渐变）

**验证：**
```bash
python main.py --test  # 测试模式查看愚者
python -m pytest tests/ -v
```

**预计工时：** 2-3 小时

---

### 阶段 2：大阿尔克纳热门牌（5张）

**优先级：** ⭐⭐⭐ 高

**原因：**
- 抽牌概率高，用户经常看到
- 这些牌画面元素丰富，优化后效果明显

**优化方向：**
1. 统一调色板，增加过渡色
2. 增加符号细节（无限符号、月亮、爱心等）
3. 改善人物比例和姿态

**文件：**
- `tarot/cards/major_arcana_part1.py`（牌1-11）
- `tarot/cards/major_arcana_part2.py`（牌12-21）

**验证：** 每优化一张牌后运行测试

**预计工时：** 10-15 小时（每张牌2-3小时）

## 颜色系统升级方案

### 当前调色板（愚者示例）

```python
FOOL_COLOR_MAP = {
    '.': '#0a111d',  # void/black
    'W': '#f0e0b0',  # cream/white highlight
    'g': '#dab772',  # gold (main)
    'G': '#a58040',  # dark gold
    'R': '#8b2500',  # dark red (rose)
    'S': '#f0d0a0',  # skin/peach
    'B': '#506468',  # blue-gray
    't': '#506468',  # teal (same as B)
    'Y': '#ffdc78',  # bright sun gold
}
```

### 建议扩展调色板

```python

    # 背景/阴影

    # 高光/白色
    'W': '#f0e0b0',  # warm white

    # 皮肤

    's': '#d0a070',  # dark skin

    # 金色
    'Y': '#ffdc78',  # bright gold
    'y': '#dab772',  # medium gold
    'G': '#a58040',  # dark gold
```

## 风险与回滚

### 可能的风险

1. **颜色映射错误** — 某些字符对应颜色不对
2. **分辨率不匹配** — 牌面宽度不是32的倍数
3. **逆位显示异常** — 旋转后画面错位

### 回滚方案

```bash
# 回滚到优化前
git checkout v0.7.6-pre-optimize

# 回滚到某个阶段
git checkout v0.7.7-fool

# 查看某个版本的牌面
git show v0.7.7-fool:tarot/cards/fool.py
```
